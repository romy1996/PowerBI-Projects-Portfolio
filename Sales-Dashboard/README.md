📊 Sales Analytics Dashboard (Power BI)
🔍 Project Overview

This project focuses on analyzing sales data to uncover key business insights related to revenue, product performance, and regional trends. The dashboard provides an interactive view of sales performance to support better decision-making.

🛠 Tools Used
Power BI
Excel / CSV
📈 Key Analysis Performed
Data cleaning and transformation using Power Query
Data modeling and relationship building
Creation of interactive dashboards and visualizations
📊 Key Insights
Identified top-selling products and categories
Analyzed sales trends over time
Evaluated regional performance and revenue contribution
Highlighted key drivers impacting overall sales
💡 Business Impact

This dashboard helps stakeholders:

Monitor sales performance
Identify growth opportunities
Make data-driven business decisions
